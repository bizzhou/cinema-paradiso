What is the project

This is a Rotten Tomato like web application. User can singup and post reviews, rate movies, and many more functionalities. For the front end, we use Angular and modularize the code, and for the back end, we use Spring Boot to create Restful API for the backend to access the data in the database. The reason we want separate front end and back end is because this we the front end developer can develop with dummy server. While the back end developer can develop with dummy requests using Postman. Separating the font end and back end speeds up the development cycle
